## 🚀 طرق التشغيل

### 1. التشغيل السريع
```bash
cd mubdi-ai
./start.sh
```

### 2. التشغيل اليدوي
```bash
# الواجهة الخلفية
cd backend
python main.py

# الواجهة الأمامية (في terminal آخر)
cd frontend
npm run preview -- --host 0.0.0.0 --port 12001
```

### 3. التشغيل بـ Docker
```bash
docker-compose up --build -d
```
